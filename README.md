# موقع أوس كبدا

## هيكل المشروع
```
ousskebda/
├── index.html        الصفحة الرئيسية (كل المحتوى والأقسام)
├── css/
│   └── style.css      كل الأنماط والتصميم
├── js/
│   └── script.js       منطق الحجز، العداد، لوحة الإدارة، الرسائل
└── assets/             (فاضي حالياً - لو رغبت تستخدم صور محلية بدل روابط الإنترنت)
```

## التشغيل
افتح `index.html` مباشرة في أي متصفح — يعمل فوراً بدون سيرفر أو تثبيت.

## النشر
انسخ المجلد كاملاً (بمجلداته الثلاثة) إلى استضافتك، مع الحفاظ على نفس هيكل المجلدات لأن `index.html` يربط الملفات بمسارات نسبية (`css/style.css` و `js/script.js`).

## التعديل السريع
- **الألوان والخطوط:** أعلى ملف `css/style.css` داخل `:root`.
- **النصوص والأقسام:** داخل `index.html`، كل قسم معلّم بتعليق عربي مثل `<!-- نظام الحجز -->`.
- **منطق العداد والحجوزات:** كل شيء في `js/script.js`، معلّق بالعربية بالكامل.
- **الخريطة:** بدّل رابط `q=Riyadh` في وسم iframe داخل قسم الموقع بموقعك الفعلي.
- **الصور:** حالياً من Unsplash عبر الإنترنت؛ لاستخدام صور محلية ضع الصور داخل مجلد `assets/` وعدّل مسارات `src` في `index.html`.

## بيانات الحجوزات
تُخزَّن في LocalStorage بالمتصفح (لا تحتاج قاعدة بيانات):
- `oussBookingsCount` → عدد الحجوزات
- `oussBookingsData` → تفاصيل كل حجز

لوحة الإدارة (زر "لوحة إدارة الحجوزات" أسفل نموذج الحجز) تعرض الجدول، وفيها زر تصدير كنص وزر مسح الكل.
